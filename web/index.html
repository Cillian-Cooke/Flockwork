JSON Format

Drop a daily pack — { "name", "start_date": "YYYY-MM-DD", "maps": [...] } — to unlock one map per day. The 📅 button reopens the picker.

The same format is used for creating a level and for saving / loading a game. The rounds and current fields are optional — omit them for a fresh level; they are included automatically when you download a saved game.

  • name — map title (first 20 chars shown)
  • grid — 2-D array of cell strings: terrain id + optional entity letter.
    "0" void (fall = die) · "1" grass · "2" wall · "1A" hero · "1s" sheep · "1a" enemy
  • scripts — loop of action tokens for each enemy / sheep
  • rounds (save) — array of completed round move arrays
  • current (save) — moves queued for the next round

Terrain: 0 void · 1 grass · 2 wall · 3 lava · 4 slip · 5 skip · 6 portal · 7 dup · 8 push · 9 repeat · 10 glide · 11 ward · 12 warp · 13 mirror · 14-100 grass

Tokens: move w a s d · attack t f g h · wait .
e = instant ability · r = charge — arms on press, fires on whichever move key comes next (stays armed through waits/attacks in between)

Heroes/enemies: uppercase = hero, lowercase = enemy — K knight · R rogue · M mage · B brute (moves 2 tiles) · H hunter

{
  "name": "My Map",
  "grid": [
    ["0",  "1",  "1A", "1",  "0" ],
    ["1",  "1",  "1",  "1",  "1s"],
    ["0",  "1a", "1",  "1",  "0" ]
  ],
  "scripts": { "s": ["w","s","."], "a": ["d","a"] },
  "rounds":  [["w","d","d"],["d","d","d","d","d"]],
  "current": []
}
